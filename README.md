# Projet Web : Le Chicaneur Numérique

Site d'informations magiques inspiré de l’univers d’Harry Potter.


## Points Techniques

- **Aucune librairie externe :** HTML5, CSS3 et JavaScript natif uniquement.
- **Sémantique :** Balises utilisées `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`.
- **CSS moderne :** Flexbox Grid et Effets lumineux (glow).
- **JavaScript Interactive :** Alerte pour les pages sensible (spoil) comme `horcruxes.html` et `relique_de_la_mort.html`.


## Structure du Projet


index.html : Page d’accueil
casting.html : Liste des personnages et acteurs
sortilèges.html : Sortilèges les plus connus
horcruxes.html : Présentation des 7 Horcruxes
relique_de_la_mort.html : Page dédiée aux 3 Reliques de la Mort
contact.html : Formulaire de contact
styles.css : Style global du site
assets/ : Image / Logo / icônes
casting_film/ : Tous les films en .html 



## Design & Direction Artistique

- **Couleur :** Doré, Sombre magique et Clair
- **Style visuel :**  Ambiance inspirée de l'hunivers d'Harry Potter / Icônes et images personnalisées, toutes dans une direction artistique cohérente / Mise en page immersive
